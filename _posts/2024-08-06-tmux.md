# Tmux usage

some simple command to use:

create session
```shell
tmux new -s {session-name}
```

split window
```
ctrl+b, %  (left-right)
ctrl+b, "" (up-down)
```

close-sub-window
```
ctrl+b, x
```

leave session
```
ctrl+b, d
```

re enter session
```shell
tmux new -t {session-name}
```
